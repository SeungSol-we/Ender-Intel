# Ender-Intel

마인크래프트라는 게임 속 엔더 크리스탈이라는 아이템의 모습에 더해 똑똑한 
ai비서로 유용함을 더한 스마트 조명입니다.

## 기능

### 1. 밸런싱 기능
- 사각형의 꼭짓점으로 균형을 잡아 서 있을 수 있음
- 밸런싱을 유지하며 제자리 회전
- 휴대폰을 이용해 모서리 균형, 꼭짓점 균형, 회전 등... 모드 변경 가능
### 2. 조명 기능
- 바닥부분과 큐브 안쪽 부분에 LED가 설치되어 있어 빛을 낼 수 있음
- 마찬가지로 휴대폰을 이용한 조명 색상변경 가능
### 3. AI 비서 기능
- 밭침대 부분(밸런싱 큐브와는 따로)에 마이크와 스피커가 장착되어 있어
  질문을 말하면 AI가 대답해주는 비서 기능이 있음.
